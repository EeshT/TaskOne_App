About TaskOne App

MyTodo is a simple yet powerful task management application built using Flask, SQLite, and Bootstrap 5. It's designed to help users manage their daily tasks efficiently.

🔧 Key Features

  • 📝 Add Todos: Easily add tasks with a title and description.
  • 📜 View Todos: All your tasks are displayed in a structured table format.
  • 🗑️ Delete Tasks: Remove any task once it is completed or no longer needed.
  • ✏️ Update Tasks: Edit an existing task’s title or description with ease.
  • 🔍 Real-Time Search: Instantly filter todos by typing in the search bar — no page reloads!
  • 🚫 Empty Entry Protection: Prevents creation of todos with empty titles or descriptions using both client-side and server-side validation.

🚀 How It Works

  1. User lands on the home page and sees the todo form and list.
  2. They can add a task by entering a title and description.
  3. Todos are stored in a database and listed in a Bootstrap-styled table.
  4. Tasks can be updated or deleted with one click.
  5. The search bar allows filtering of tasks
